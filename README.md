# 💌 Elegant Wedding Invitation Design – Golden Bloom Theme

## 📌 Project Overview  
This project showcases a beautifully crafted wedding invitation design featuring a soft pastel background, delicate floral elements, and luxurious gold accents. The design focuses on elegance, readability, and a premium visual experience.

---

## 🎯 Objective  
To design a modern and visually appealing wedding invitation that reflects sophistication, romance, and celebration.

---

## 🎨 Design Features  
- Elegant floral frame with gold detailing  
- Soft blush watercolor texture  
- Clean and balanced typography  
- Front & back card layout  
- Print-ready high-resolution design  

---

## 🛠️ Tools Used  
- Canva  
- Figma  
- Adobe Illustrator  

---

## 📂 Deliverables  
- Wedding Invitation Card (Front Design)  
- Wedding Invitation Card (Back Design)  
- Social Media Version (Instagram Story/Post)  
- High-quality Printable PDF (300 DPI)  

---

## 💡 Design Principles Applied  
- Typography hierarchy  
- Color harmony  
- Minimal and clean layout  
- Visual balance and spacing  

---

## 🚀 Use Case  
Perfect for:  
- Wedding Invitations  
- Engagement Cards  
- Premium Event Invitations
